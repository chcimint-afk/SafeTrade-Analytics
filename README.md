# 🚀 SafeTrade Analytics — Multi-Language Workspace Directory

Welcome to the **SafeTrade Analytics** workspace. This repository contains the complete configuration, documentation, specifications, and source code for an ultra-premium, dark-themed, and highly secured personal trading dashboard.

This project is a **personal, non-commercial, and educational tool** designed to optimize individual trade execution, manage capital under strict risk-management parameters, and visualize market signals safely.

---

## 🇷🇺 Русский (Russian)

### 📌 Описание Проекта
**SafeTrade Analytics** — это персональный интеллектуальный помощник для трейдеров, созданный для индивидуального использования (в том числе на мобильных устройствах). Проект полностью ориентирован на строгий контроль рисков (правило 1% риска на сделку) и защиту личного капитала.
*   **Статус:** Личный, некоммерческий проект.
*   **Цель отчетов и симуляций:** Все бэктесты и симуляции в папке `/specs/reports` предназначены исключительно для проверки математической точности торговых стратегий на исторических данных и не являются коммерческим продуктом.

### 📁 Структура Репозитория
*   **`/plan`** (Корневая папка планов): Содержит генеральный план распределения капитала («Мастер-Трейдер») и пошаговый план разработки MVP.
*   **`/specs`** (Технические спецификации): Содержит правила архитектуры, протоколы безопасности (Anti-Algo Shield, Profit Shield) и полную спецификацию потоков данных (Data Flow).
*   **`/app-code`** (Исходный код): Реализация интерфейса и логики приложения на стеке Next.js, React и Tailwind CSS.
*   **`/.agents` & `AGENTS.md`** (Конфигурация ИИ): Описание навыков и правил для Вашего ИИ-разработчика.
