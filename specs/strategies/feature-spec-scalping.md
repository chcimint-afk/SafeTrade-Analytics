# Feature Specification: Low-Risk Scalping Signals

## 1. Overview
Функция **Low-Risk Scalping** предназначена для поиска быстрых сделок на коротких временных отрезках (1-5 минут) для получения микро-прибыли, которая в сумме дает стабильный дневной доход.

## 2. Logic & Technical Indicators
Сигналы генерируются при совпадении трех условий:
1. **RSI (Relative Strength Index):** Перепроданность (<30) или перекупленность (>70).
2. **Moving Averages (EMA 20/50):** Подтверждение тренда.
3. **Volatility Check:** Наличие достаточного движения (не "флэт").

## 3. Safety Rules (Hardcoded)
- **Stop-Loss:** Обязателен для каждой сделки. Расстояние до стоп-лосса определяет объем лота.
- **Take-Profit:** Устанавливается на уровне 1.5x или 2x от риска (Ratio 1:2).
- **Max Trades:** Не более 3-5 прибыльных сделок в день.

## 4. UI Representation
- Зеленая/Красная зона уверенности.
- Кнопка "Copy to Clipboard" для параметров сделки.
- Звуковой сигнал при появлении "Золотого сигнала" (вероятность >80%).
