# Functional Map: SafeTrade Analytics

## 1. App States & Screens
- **Dashboard (Главный экран):** 
  - Текущий баланс и дневная цель (например, "Осталось заработать 5$ до цели 1%").
  - Рекомендация: "Торговать" или "Отдыхать".
- **Signals Panel (Панель сигналов):**
  - Список пар (BTC/USDT, EUR/USD).
  - Индикатор "Сила сигнала" (на основе RSI, MACD).
  - Точки входа, Стоп-лосс и Тейк-профит.
- **Risk Calculator (Калькулятор):**
  - Поле для ввода баланса.
  - Автоматический расчет лота/объема сделки исходя из риска 1%.

## 2. Core Components
- **Signal Provider:** Модуль, анализирующий технические индикаторы.
- **Position Tracker:** Мониторинг открытых сделок и уведомление о закрытии по цели.
- **Educational Guide:** Всплывающие подсказки о том, почему этот сигнал считается безопасным.

## 3. User Flow
1. Пользователь заходит в приложение и вводит свой баланс (например, 1000 EUR).
2. Приложение рассчитывает безопасный размер сделки (например, риск 10 EUR).
3. Приложение сканирует рынок и выдает сигнал: "EUR/USD — Покупка (Вероятность 75%)".
4. Пользователь совершает сделку и закрывает её при достижении микро-прибыли.
5. При достижении дневной цели (10-15 EUR) приложение советует закрыть терминал.
