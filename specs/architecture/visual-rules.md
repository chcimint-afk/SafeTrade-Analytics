# 🎨 Visual Rules / Skills (Визуальные спецификации)

## 📌 Дизайн-система SafeTrade Analytics

Данный документ описывает визуальный стандарт и принципы построения пользовательского интерфейса панели управления SafeTrade Analytics. Основная цель — создание премиального, строгого и информативного интерфейса в стиле финансового терминала Bloomberg.

---

## 🎨 1. Цветовая палитра (Color Tokens)

Дизайн базируется на глубоких темных тонах с неоновыми/акцентными индикаторами состояний торговых сессий.

*   **Основной фон (Background):** `Dark Charcoal` (`#0B0E11` / HSL `210, 18%, 5%`) — глубинная темная тема для снижения нагрузки на глаза при длительной работе.
*   **Вторичный фон панелей (Card Background):** `Glass Dark` (`rgba(20, 26, 33, 0.65)` / HSL `214, 25%, 10%`) с применением эффекта размытия заднего плана (glassmorphism).
*   **Акцентный цвет (Primary/Brand):** `Safe Cyan` (`#00F0FF` / HSL `180, 100%, 50%`) — используется для ключевых показателей, активных состояний и интерактивных переключателей.
*   **Цвета индикации (Status Colors):**
    *   **Прибыль / Long (Success):** `Neon Emerald` (`#00E676`) — для положительного PnL, роста и включенных функций.
    *   **Убыток / Short (Danger):** `Neon Rose` (`#FF1744`) — для отрицательного PnL, рисков и критических ошибок.
    *   **Предупреждения / Ожидание (Warning):** `Amber Glow` (`#FFD600`) — для неактивных/приостановленных модулей.

---

## ✍️ 2. Типографика (Typography)

Для обеспечения идеальной читаемости финансовых данных используются современные шрифты с высокой четкостью начертания цифр:

*   **Основной шрифт (Sans-Serif):** `Inter` или `Outfit` (подключается через Google Fonts).
*   **Шрифт для цифр и таблиц (Monospace):** `JetBrains Mono` или `Roboto Mono` для табличных данных, чтобы цифры не смещались при изменении значений.
*   **Иерархия размеров:**
    *   `H1` — 32px (Заголовки экранов)
    *   `H2` — 20px (Заголовки секций/модулей)
    *   `Body` — 14px (Стандартный текст)
    *   `Data/Caption` — 12px (Второстепенная информация, мелкие метрики)

---

## ✨ 3. Интерактивные элементы и микро-анимации

Интерфейс должен ощущаться «живым» и мгновенно реагировать на действия пользователя:

*   **Эффект стекла (Glassmorphism):**
    *   Панели имеют границу: `1px solid rgba(255, 255, 255, 0.08)`.
    *   Эффект размытия: `backdrop-filter: blur(12px)`.
*   **Интерактивные состояния (Hover/Active):**
    *   Кнопки и ссылки должны плавно менять прозрачность или цвет фона (`transition: all 0.2s ease-in-out`).
    *   Эффект свечения (glow): для активных кнопок автопилота используется мягкая тень `box-shadow: 0 0 15px rgba(0, 240, 255, 0.3)`.
*   **Переключатели (Toggles):**
    *   При включении автопилота или защитных систем переключатель плавно перемещается, меняя цвет с серого на ярко-зеленый (`Neon Emerald`).
