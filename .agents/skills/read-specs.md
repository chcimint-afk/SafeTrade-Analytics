# AI Skill: Read Specifications

## Description
Этот навык позволяет ИИ (Агенту) динамически находить и читать спецификации проекта (specs), не загружая их все одновременно в память. Это предотвращает перегрузку контекста.

## Instructions
1. Все спецификации проекта лежат в директории `specs/`.
2. НИКОГДА не пытайся угадать требования к проекту или выдумать их.
3. Если пользователь просит тебя написать код, реализовать фичу или задает вопрос по проекту, используй инструменты просмотра файлов (`view_file`, `list_dir`), чтобы прочитать соответствующий документ в папке `specs/`.
4. Категории файлов:
   - Если вопрос касается архитектуры, стека технологий или структуры папок -> читай `technical-spec.md`.
   - Если вопрос касается бизнес-логики, алгоритмов или глобальных целей -> читай `global-spec.md`, `autopilot-trading.md`.
   - Если вопрос касается конкретной страницы, кнопки или интерфейса -> ищи подходящий файл в `specs/` (например, `functional-map.md` или `feature-spec-*.md`).
5. Прочитав нужный файл, используй полученные данные для выполнения задачи.
